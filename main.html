Question-2: What is the difference between innerHTML, innerText, and textContent ?

An innerHTML return Returns or sets the HTML content inside an element, including tags and formatting. and an innerText return Returns only the visible text inside an element . that is the difference between innerHTML and innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where instead of adding event listeners to individual child elements, you assign a single event listener to a parent element. This works because events bubble up from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates up through its ancestors in the DOM tree until it reaches the document object.
